# Iceberg Case Frontend

Nuxt 3 + Pinia + Tailwind CSS ile yazılmış emlak yönetim paneli.

![Iceberg Case](public/project-image.png)

## Kurulum

```bash
bun install
```

## Çalıştırma

```bash
bun run dev
```

Uygulama `http://localhost:7979` adresinde açılır.

## Kullanılmayan Kod Analizi

```bash
bunx knip
```
